## Fonctionnalités

L'application propose plusieurs niveaux d'analyse :

* **Aperçu des données :** Exploration brute et statistiques descriptives du dataset.
* **Analyse Géographique :** Focus spécifique sur la distribution des salaires en **France**
* **Comparaisons Interactives :** Graphiques en barres pour comparer les moyennes selon le titre du poste, le type de contrat ou la localisation.
* **Évolution Temporelle :** Suivi de la progression des salaires pour les 10 métiers les plus fréquents au fil des années.
* **Corrélations :** Matrice de corrélation pour comprendre les liens entre les variables numériques.
* **Filtres Dynamiques :** Recherche personnalisée par plage de salaire, niveau d'expérience et taille d'entreprise.
* **Impact du Télétravail :** Visualisation de l'influence du mode de travail sur la rémunération.

## Installation et Lancement

Si vous souhaitez lancer le projet localement :

1.  **Cloner le dépôt :**
   
    git clone 
    cd votre-repo
    

2.  **Installer les dépendances :**
    
    pip install -r requirements.txt
    

3.  **Lancer l'application :**
    
    streamlit run application.py
